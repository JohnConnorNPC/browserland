A sticky note is a small, always-visible scratch pad for quick text. Notes use a fixed-width font, so pasted code, ASCII, and aligned columns line up the way they do in the terminal and editor. Notes stay out of the taskbar by default.

## Notes on the taskbar

Prefer a taskbar button per note? Turn on **show sticky-note windows on the taskbar** under **Control Panel → Mods** (the *Sticky notes* setting). Every open note then gets a taskbar button like any other window — click to focus or restore, click the active one to minimize — and the button tracks the note's title and color. Flipping the setting applies immediately to notes already open. The default (off) keeps today's behavior.

## Reopening a closed note

If you close a sticky note that still has text in it, it is not lost: a non-empty closed note is listed under a **Closed notes** section at the bottom of the **+** (launch) menu. Click it there to reopen it. An empty note you close is simply discarded. For more on the launch menu, see [[Taskbar]].
