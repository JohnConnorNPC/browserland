- [x] [F022] Host file API (auth-gated) ‚Äî `/file/*` read/write/list across the FULL host (intentional full-host-authed model #35, NOT a rooted path sandbox; `editor_root` is only the empty-path default), gated token-or-loopback (token‚Üí401 auth_required / no-token non-loopback‚Üí403 disabled_no_token, loopback by sockname); list `{root,cwd,parent,entries[{name,type,size}]}`, read UTF-8/`content_b64` (non-utf8‚Üí400 not_utf8), atomic write, 5 MiB cap (too_large), ADS/half-absolute path‚Üíbad_path; backs editor F046 + file manager F047 (webterm/broker/app.py)
- [x] [F077] Synced UI-state API ‚Äî `GET/PUT /state` shared per-broker settings+layout blob with optimistic-concurrency `rev` (409 conflict inlines the live state; lease-gated PUT returns not_active for a non-active browser); durable self-healing `webterm_state.json` (atomic write, rev persisted across restarts); same token-or-loopback gate as /file/* (webterm/broker/app.py)
- [x] [F078] Browser-facing `GET /profiles` ‚Äî names-only launchable profiles (default + list; command/cwd never leave the broker), same token-or-loopback gate as /sessions; backs the launch (+) menu, per-host remote profile groups, the default-profile picker, and "Open in folder" (distinct from /mcp/profiles) (webterm/broker/app.py)
